### 第一：选择排序

选择排序是最直观，最简单的一种排序算法，其每次从待排序的序列中取出最大 或者最小的一组元素放在序列的起始位置，排序的时间复杂度为：O\(n^2\)，空间复杂度：O\(1\)

如待排序的序列为：9，5，8，0，2，1

```
        第一轮过后：0，5，8，9，2，1

        第二轮过后：0，1，8，9，2，5

        第三轮过后：0，1，2，9，8，5

        第四轮过后：0，1，2，5，8，9

        第五轮过后：0，1，2，5，8，9
```

```
NSMutableArray *array = [NSMutableArray arrayWithObjects:@9,@5,@8];
```
